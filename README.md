Excel Sales Analysis Dashboard (FNP Dataset)

Project Overview

This project focuses on building a business-ready Sales Analysis Dashboard in Microsoft Excel to track performance and identify key revenue drivers across products, categories, cities, and occasions. The dashboard is designed for quick decision-making with interactive filters and clear KPI tracking.

Key Business Metrics :

- Total Revenue: 35,20,984

- Total Orders: 1000

- Average Order to Delivery Time: 5.53 days

- Average Customer Spending: 3,520.98

Dashboard Features :

- Revenue trend analysis by month and weekday

- Top-performing products by revenue

- Top cities by order volume

- Revenue breakdown by category and occasion

- Interactive filters for order date, delivery date, and occasion

Tools and Skills Used :

- Microsoft Excel

- Pivot Tables and Pivot Charts

- Slicers and Filters

- KPI Cards and Dashboard Layout Design

- Data Cleaning and Business Insights

Outcome :

This project improved my ability to convert raw sales data into actionable insights using Excel, with a focus on KPI monitoring, visualization, and business storytelling.
